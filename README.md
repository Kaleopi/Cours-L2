# Cours-L2

Commande pour cloner :
	- git clone https://github.com/Kaleopi/Cours-L2.git

Envoyer ses dossier :
	- Ajouter les dossiers, puis dans un cmd à l'emplacement du dossier :
		- git add .
		- git commit -m "informations sur la modification"
		- git push -v (pour vérifier que tout s'envoie bien)

Mettre à jour les dossiers quand quelqu'un d'autre l'a mis à jour :
	- git pull
