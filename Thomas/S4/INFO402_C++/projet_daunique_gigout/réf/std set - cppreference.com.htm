[edit]
 
Defined in header <set>
template<

    class Key,
    class Compare = std::less<Key>,
    class Allocator = std::allocator<Key>

> class set;

std::set est un conteneur associatif qui contient un ensemble trié d'objets uniques de type Key. Le tri est effectué en utilisant Compare, une fonction de comparaison de Key. Les opérations de recherche, de retrait et d'insertion ont une complexité logarithmique. Les std::set sont généralement mis en œuvre aux travers d'arbres rouge-noir.

std::set répond aux exigences de Container, AllocatorAwareContainer, AssociativeContainer et ReversibleContainer.
