[edit]
 
Defined in header <set>
template<

    class Key,
    class Compare = std::less<Key>,
    class Allocator = std::allocator<Key>

> class multiset;
Multiset est un conteneur associatif qui contient un ensemble d'objets triés clé de type. Contrairement ensemble, plusieurs touches avec des valeurs égales sont autorisés. Le tri est fait en utilisant la fonction de comparaison Comparer clé. Opérations de recherche, d'insertion et le retrait ont une complexité logarithmique .
Original:
Multiset is an associative container that contains a sorted set of objects of type Key. Unlike set, multiple keys with equal values are allowed. Sorting is done using the key comparison function Compare. Search, insertion, and removal operations have logarithmic complexity.
